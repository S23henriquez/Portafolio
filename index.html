// Web App · PHP · MySQL

Sistema de Gestión de Taller

Aplicación web completa desarrollada en PHP con MySQL para gestionar clientes, vehículos, facturas y presupuestos de un taller real. Usada activamente en producción.

  • Gestión de clientes con múltiples vehículos asociados
  • Creación de facturas y presupuestos con catálogo de servicios
  • Aplicación de descuentos y cálculo automático de IVA
  • Generación de PDFs profesionales con todos los detalles
  • Sistema de autenticación con roles y contraseñas encriptadas
  • Panel de administración con historial completo

Proyecto personal que sigo mejorando activamente. Primer proyecto desplegado en producción real.

// Android · Kotlin

Penaltis Multiplayer

Juego Android multijugador que simula un partido de penaltis entre dos jugadores (Host y Guest) con lógica de turnos y resultado final.

  • Comunicación entre jugadores en tiempo real
  • Sistema de turnos y control de marcador
  • Interfaz animada con efectos visuales
  • Gestión de estados: victoria, derrota y empate

Proyecto clave para reforzar lógica de juego, control de estados y comunicación entre dispositivos Android.
